# Produits – Structure des images

Pour chaque produit, placez vos images dans le dossier correspondant :

| Produit | Dossier |
|---------|---------|
| Distributeurs de carburant | `distributeurs-carburant/` |
| Tuyauterie KPS - OPW | `tuyauterie-kps-opw/` |
| Raccordements Ridart | `raccordements-ridart/` |
| Flexible et Pistolet | `flexible-pistolet/` |
| Téléjaugeage (ATG) | `telejaugeage-atg/` |
| Système FMS | `fms/` |
| Solutions de Paiement | `solutions-paiement/` |
| Citernes & Stockage | `citernes-stockage/` |
| Chambres en composite | `chambres-composite/` |
| Car Wash | `car-wash/` |
| Équipements Servicing MECLUBE | `equipements-servicing-meclube/` |
| Pompes immergées Barbero | `pompes-immergees-barbero/` |

**Noms des fichiers :** `img-01.jpg`, `img-02.jpg`, `img-03.jpg`, etc.  
Formats acceptés : `.jpg`, `.jpeg`, `.png`
